# WNCMS

![WNCMS Dashboard](/documentations/images/wncms-dashboard.webp)

WNCMS 是為了解決創作者、開發者與商家在建站過程中遇到的實際問題，而打造的一套現代化、多語系、高擴充性的 CMS。系統基於 Laravel 12 開發，採用乾淨的架構、模組化設計以及可擴充的生態，能滿足 2025 年以後的網站與系統需求。

WNCMS 的設計目標，是在以下幾點之間取得最理想的平衡：

- 初學者也能輕鬆上手  
- 開發者擁有足夠的彈性與擴充能力  
- 正式網站運行時保持高效能  
- 設計師與接案公司能快速自訂、快速交付  

無論是部落格、企業官網、著陸頁系統、多網域平台，甚至是需要自訂資料模型的應用，WNCMS 都能提供穩定且可預期的基礎，避免許多傳統 CMS 的限制。

---

## WNCMS 為什麼會誕生

WNCMS 最初是我自用的專案，起因於市面上找不到符合需求的系統。我需要的是：

- 基於 Laravel 的乾淨、現代化架構  
- 能同時支援多語系、多網站、多網域  
- 一個比 WordPress 更易於維護與理解的主題框架  
- 穩定、安全且可維護的外掛系統  
- 能作為 SaaS 底層的 CMS  
- 一個後台能統一管理多種自訂資料模型（文章、連結、選單、廣告等）  

市面上的工具要嘛太臃腫、要嘛不易擴充，也有許多並不適合長期維護或多人協作。因此我決定從零開始設計一套可預期、可擴充又易用的 CMS。

如今，WNCMS 已經在多個正式網站、商用主題、SaaS 專案與影音平台上穩定運行。

## WNCMS 提供什麼

### 給一般使用者

- 清晰易懂的後台介面  
- 可安裝或自訂的前台主題  
- 文章管理  
- 連結管理  
- 廣告管理  
- 選單編輯  
- 完整多語系內容支援  

### 給開發者

- Laravel 12 生態完整支援  
- Manager 架構資料模型  
- Models / Controllers / Views 可覆寫  
- 主題開發簡單明確  
- 外掛開發輕量易懂  
- 快速生成 API  
- 可翻譯模型  
- 統一快取與查詢流程
